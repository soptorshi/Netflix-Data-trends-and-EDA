# Netflix Data Trends and Explanatory Data Analysis
• Dataset source: Netflix titles (Kaggle)

• Objective: Analyze the Netflix titles dataset to uncover trends in content production, distribution, genres, and duration using Python and data analysis techniques.

• Tools: Python (Pandas, Numpy), Tableau

• Key insights: Netflix content is dominated by international movies and dramas, with the majority of titles released after 2015, and the United States and India are contributing the largest share of content.
